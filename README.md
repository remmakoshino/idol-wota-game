# アイドルヲタク逃走ゲーム 🎤🎮

マインクラフト風のキャラクターで遊ぶ、アイドルライブ厄介ヲタク3Dアクションゲーム

## 🎮 ゲームの特徴

### メインゲーム
- **3つのステージ**: ライブハウス、ホール、アリーナ（難易度別）
- **マインクラフト風キャラクター**: ブロック状のキュートなデザイン
- **5人のアイドルグループ**: それぞれ異なる衣装カラー
- **4種類の厄介行為**: 各キーで異なるアクション
  - 💥 **E**: モッシュ
  - ⬆️ **Q**: リフト（2人のヲタクが持ち上げてくれる）
  - 🍺 **F**: 酒缶投げ（3Dで缶が飛んでいく）
  - 📢 **R**: 野次（ランダムで4種類のメッセージ）
- **パーティクルエフェクト**: アクションに応じた視覚効果
- **逃走システム**: マッチョなセキュリティスタッフから逃げる
- **リアルな3D環境**: ステージごとに異なるライブ会場を再現

### ミニゲーム
- **直缶チャレンジ**: 缶の下に穴を開けて一気飲みの速さを競う
- **ストロング酒雑魚**: オリジナル缶デザイン
- **リアルな飲酒演出**: 頭、口、手、缶のイラスト付き
- **イッチャン好き！💖**: 完飲時の特別メッセージ

## 🛠️ 技術スタック
- **Three.js** (v0.158.0): 3Dグラフィックス
- **React Three Fiber** (v8.15.11): Reactベースの3Dレンダリング
- **@react-three/drei** (v9.92.7): 3Dヘルパーコンポーネント
- **Vite** (v5.0.8): 高速ビルドツール
- **TypeScript** (v5.2.2): 型安全な開発
- **カスタム物理エンジン**: 独自の重力・衝突判定システム

## 🎯 操作方法

### 基本操作
- **WASD**: 移動
- **Shift**: ダッシュ
- **Space**: ジャンプ
- **マウス**: 視点操作（クリックでポインターロック）

### 厄介行為
- **E**: モッシュ 💥 (+100点)
- **Q**: リフト ⬆️ (+150点) - 他のヲタクが持ち上げてくれる
- **F**: 酒缶投げ 🍺 (+200点) - 実際に缶が飛んでいく
- **R**: 野次 📢 (+120点) - ランダムで以下のメッセージを表示
  - 「男がいるなら、謝罪しろ〜！」
  - 「責任から逃げるな！」
  - 「メン地下彼氏は？」
  - 「風俗に在籍ある？」

### ミニゲーム
- **Space連打**: 直缶を一気飲み

## 🎲 ゲームの遊び方

### メインゲーム
1. **メインゲーム開始**をクリック
2. **ステージを選択**（ライブハウス/ホール/アリーナ）
3. 画面をクリックしてポインターロックを有効化
4. WASDキーで移動、Shiftでダッシュ、Spaceでジャンプ
5. **厄介行為を実行**してスコア獲得：
   - **E**: モッシュ
   - **Q**: リフト（2人のヲタクが出現）
   - **F**: 酒缶投げ（缶が飛んでいく）
   - **R**: 野次（4種類からランダムでメッセージ表示）
6. マッチョなセキュリティスタッフから逃げ切れ！
7. 捕まったらゲームオーバー

### ステージ難易度
- 🏠 **ライブハウス** (易): 2人のガード、小規模
- 🏛️ **ホール** (普通): 4人のガード、中規模
- 🏟️ **アリーナ** (難): 6人のガード、大規模

### 直缶チャレンジ
1. **直缶チャレンジ**をクリック
2. スタートボタンを押す
3. **スペースキーを連打**して缶を一気飲み
4. 最速タイムを目指せ！
5. 完飲すると「イッチャン好き！💖」

## 💡 技術的な特徴

### キャラクターシステム
- **マインクラフト風モデル**: BoxGeometryで構成されたブロック状キャラクター
- **3種類のキャラクター**:
  - プレイヤー: 歩行アニメーション付き
  - セキュリティガード: マッチョな体格、サングラス、バッジ付き
  - アイドル: 5人、異なる衣装カラー、ツインテール

### ゲームシステム
- **動的ステージシステム**: 3種類のステージ設定
- **パーティクルシステム**: 缶投げ、モッシュ、捕獲時のエフェクト
- **物理シミュレーション**: カスタム重力・衝突判定（外部ライブラリなし）
- **3D投擲システム**: 放物線軌道で缶が飛ぶ
- **AIチェイス**: セキュリティスタッフの自動追跡
- **スコアシステム**: 行為ごとに異なるポイント
- **リフトメカニクス**: 一時的に高度上昇、サポートキャラ出現

### ビジュアル
- **リアルタイムレンダリング**: Three.jsによる3D描画
- **動的照明**: ポイントライト、スポットライト
- **パルスアニメーション**: 野次メッセージのエフェクト
- **CSS統合**: UIアニメーション

### デプロイ手順

1. **GitHubリポジトリにプッシュ**
```bash
git remote add origin https://github.com/YOUR_USERNAME/idol-wota-game.git
git push -u origin main
```

2. **GitHub Pages設定**
   - リポジトリの **Settings** → **Pages**
   - **Source**: **GitHub Actions** を選択
   - 自動的にビルド・デプロイが開始されます

3. **更新をデプロイ**
```bash
git add .
git commit -m "Update game features"
git push
```

GitHub Actionsが自動的にビルドしてデプロイします！
